## Track Record

### 🏙️ Amsterdam Municipal Data (2017–2020)

Built data infrastructure powering [data.amsterdam.nl](https://data.amsterdam.nl):

- **parkeerscans** — Mapping parking pressure through scan data
- **afvalcontainers** — Underground garbage disposal bins
- **bag_services** — Building Address Registry (BAG) API
- **handelsregister** — Handelsregister (HR) data API
- **parkeervakken** — Parking lot data with REST API

### 🗺️ Geo-Spatial & Vector Tiles

- **Tegola** — Mapbox Vector Tile server in Go (contributor)
- **pg_tileserv** — Thin PostGIS-only tile server (contributor)
- **vector-inspector** — Visual inspection of vector tile sources
- **openmaptiles** — Vector Tile Schema Implementation (contributor)

### 📊 Public Data Platforms

- **DEGO** — Energy transition data platform ([dego.vng.nl](https://dego.vng.nl))
- **DOOK** — Public safety data platform ([dook.vng.nl](https://dook.vng.nl))
- **Signalen.org** — Municipal issue reporting system for the Netherlands
- **NLX** — Inter-organisational federated authentication for API landscapes

### 🔧 Open Source

- **python-snap7** — Co-created the open-source PLC communication library. ~50 million downloads for a niche industrial automation problem.
- **lambda_db** — In-memory database with HTTP API (Go)
- **csvpgvoer** — Feed CSV data into PostgreSQL with row validation (Go)
